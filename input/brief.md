# Jeux

Faire un jeu space invaders.

## Contraintes

- Le faire en techno web
- back-end et front end

- Front-end reactjs
- Back-end express. BDD fichiers

- Front-end avec effet sonore, et programmation en WebGL 3D avec librairie si besoin.
- On mettra l'accent sur un bonne qualite visuelle du jeu.

- On fera des classements entre les joueurs (par score)
